# 🎮 Tic-Tac-Toe

A classic two-player Tic-Tac-Toe game built with HTML, CSS, and JavaScript. Play online with friends!

## 🌐 Play Online

Visit: https://wtlj2026.github.io/tic-tac-toe

## 🎯 Features

- ✨ Beautiful, responsive UI
- 🎮 Real-time two-player gameplay
- 📊 Score tracking (saved in browser)
- 📱 Mobile-friendly design
- 🔗 Easy share link for friends
- 🏆 Win detection with highlights
- 🤝 Draw detection

## 🎮 How to Play

1. Players take turns clicking on empty squares
2. Player X goes first
3. Get 3 in a row (horizontal, vertical, or diagonal) to win
4. Click "New Game" to play another round
5. Share the link with friends to play online!

## 📂 Files

- `index.html` - Game structure
- `style.css` - Styling and responsive design
- `script.js` - Game logic and interactivity

## 🚀 Deployment

This project is deployed on GitHub Pages. Any changes pushed to the `main` branch will automatically update the live site.

## 💾 Local Development

1. Clone the repository
2. Open `index.html` in your browser
3. Start playing!
